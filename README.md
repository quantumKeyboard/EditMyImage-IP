# PixelLab — ReadMe

## Running the Tool

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Structure

```
image_tool/
├── app.py              # Entry point, layout, navigation, global CSS
├── requirements.txt
└── modules/
    ├── unit1.py        # File handling, color spaces, sampling
    ├── unit2.py        # All enhancement operations
    ├── unit3.py        # Edge detection
    └── unit4.py        # Compression (JPEG + Huffman)
```
